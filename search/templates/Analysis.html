Bioinformatics Analysis

FDRdb allows you to upload your own RNA expression profile data for analysis. You should prepare two kinds of profiles:

  • Expression profile (Download example file)

  • Sample class profile (Download example file)

Profiles must be uploaded in .txt format.
You can select the species for which the data correspond to get the fibrosis-related RNA results. Point the “Apply” button, the FDRdb will perform differential expression analysis and pathway enrichment. All the results are visualized and can be downloaded. Please pay attention! If you want to get fibrosis related RNA results, the RNA label in your profiling data must follow the rules.
See the "Help" page for more details.
